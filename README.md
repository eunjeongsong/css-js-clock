# CSS 를 이용한 시계 만들기

- vanilla script + CSS
- transform, transition 속성 사용.

## CSS

### transform-origin: 100%;

- 대상 시계 바늘의의 전체 길이를 이동 대상으로 설정.

### transform: rotate(90deg);

- 바늘의 위치가 90도(정각) 부터 시작하도록 설정.

### transition: all 0.05s;

- 위의 속성들이 0.05초의 시간동안 적용되도록 설정.
